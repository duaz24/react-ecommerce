# Projeto E-commerce em React

Projeto académico desenvolvido com React, com consumo de API externa.

## Funcionalidades
- Listagem de produtos
- Detalhe de produto
- Carrinho de compras
- Navegação com React Router

## Tecnologias
- React
- JavaScript
- React Router
- FakeStore API

## Como executar
npm install  
npm start
